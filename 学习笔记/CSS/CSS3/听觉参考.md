# 听觉参考

### 听觉样式表

听觉样式表使用语音合成和声音效果的组合来使用户收听信息，而不是阅读信息。

听觉演示可用于：

- 由盲人
- 帮助用户学习阅读
- 帮助有阅读问题的用户
- 用于家庭娱乐
- 在车内
- 由印刷品阅读障碍社区提供

听觉演示将文档转换为纯文本，并将其提供给 屏幕阅读器（读取屏幕上所有字符的程序）。

听觉样式表的示例：

```css
h1, h2, h3, h4 {
  	voice-family: male;
  	richness: 80;
  	cue-before: url("beep.au")
}
```

上面的示例将使语音合成器播放声音，然后以非常丰富的男性声音朗读标题。
